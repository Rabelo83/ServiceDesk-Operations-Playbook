# Service Desk Operations Playbook 🚀

> A personal engineering lab and portfolio demonstrating enterprise endpoint management, process automation, and ITIL-aligned service delivery for Tier 3 IT operations.

## 📖 Overview

This repository serves as a centralized knowledge base, sandbox environment, and operational playbook. It is designed to demonstrate advanced IT Service Management (ITSM) workflows, endpoint provisioning, and tier-3 escalation methodologies.

Key focus areas include:
- **Mobile Device Management (MDM):** Microsoft Intune, Apple Business Manager (ABM).
- **Automation & Scripting:** PowerShell and Python scripting for remote diagnostics and deployment.
- **ITIL Service Management:** Incident, Problem, and Change Management documentation.
- **Enterprise Integrations:** Webhooks, API automations, and CRM integrations (e.g., Odoo, Twilio).

---

## 🗂️ Directory Structure

The repository is organized by operational function:

*   **`/Automations/`** - Scripts (PowerShell/Bash/Python) used for zero-touch provisioning, reporting, and proactive maintenance.
*   **`/Intune-Configs/`** - Documentation and JSON exports of compliance policies, app deployments, and conditional access rules.
*   **`/SOPs/`** - Standard Operating Procedures for Tier 2/3 Help Desk duties (e.g., Offboarding workflows, advanced troubleshooting guides).
*   **`/Projects/`** - Dedicated folders for high-impact projects (e.g., Transit SMS Tracking System, Free Fare Odoo Migration).
